# Stanford CS331B: Interactive Simulation for Robot Learning

### Description
This course focuses on the "Embodied" part of AI. It explores how high-fidelity simulations and interactive environments can be used to train robots before deploying them in the real world.

### Key Topics
* Physics Engines (Isaac Gym, SAPIEN)
* Sim-to-Real Transfer
* Large Language Models (LLMs) for Robotics
* Visual-Motor Skills
