# Stanford CS224R: Deep Reinforcement Learning

### Description
Taught by Chelsea Finn, this course focuses on the "Robot Learning" aspect of AI, emphasizing how agents can learn to make sequences of decisions in complex, uncertain environments.

### Key Topics
* Foundations of RL
* Deep RL Architecture
* Multi-task and Meta-Reinforcement Learning
* Robotic Manipulation with RL
