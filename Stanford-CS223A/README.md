# Stanford CS223A: Introduction to Robotics

### Description
The foundational course for robotics at Stanford, taught by Oussama Khatib. It provides a rigorous introduction to the mathematical modeling of robot manipulators.

### Key Topics
* Spatial Descriptions & Transformations
* Forward and Inverse Kinematics
* Jacobians: Velocities and Static Forces
* Robot Dynamics and Force Control
