# CMU 16-831: Introduction to Robot Learning

### Description
This course explores the intersection of robotics and machine learning. It focuses on how robots can learn from data to perform complex tasks, moving beyond traditional hand-coded controllers.

### Key Topics
* Imitation Learning (Behavior Cloning, DAgger)
* Online Learning & Regret Minimization
* Reinforcement Learning for Robotics
* Real-world Deployment Challenges
